2. **国际 CLI 双版本升级被低估**：Claude Code 2.1.217 与 Codex 0.145.0 同日发布，Codex `/import` 可一键迁移 Cursor/Claude Code 配置——在 Fable 5 分层定价次日，媒体普遍认为这是 OpenAI 承接 Pro 用户分流的关键动作，但国内媒体对 TRAE SOLO 的关注度显著更高。

### 分歧

1. **TRAE SOLO 的真实工程价值**：智东西（7/21）与腾讯新闻（7/21–7/22）强调「一句话交付」「十几分钟生成电商网站」的演示效果；36氪（4 月）对 Cursor 3 的「智能体控制台」持审慎态度，认为「如果开发者未来不再依赖 IDE，Cursor 根基会动摇」——对 SOLO 能否在复杂存量代码库落地，媒体立场分化明显。

2. **国产 vs 国际定价策略**：智东西报道 TRAE 首月 $3、后续 $10/月，对比 Cursor $20/月、Windsurf $15/月；36氪（4 月）指出 Trae 对个人开发者完全免费且集成多模型——⚠️ SOLO 模式需 Pro + SOLO Code 解锁，与「完全免费」叙事存在张力，媒体对实际使用成本的解读不一。
